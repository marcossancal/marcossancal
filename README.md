## 🚀 What I'm Building
### 🔷 [FlexCore](https://github.com/marcossancal/FlexCore) — *Open Source · Active Maintenance*
> **A self-hosted, low-code backend platform built in PHP.**

Think Airtable — but on your own server, with a full REST API, automation engine, plugin system, and role-based access control. With FlexCore you can create dynamic entities (tables), define 30+ field types, manage records through a web UI, and expose everything via API — **without writing a single line of SQL or backend code**.

Built for developers who want to ship MVPs fast without sacrificing control.

`PHP` · `MySQL/MariaDB` · `REST API` · `Open Source` · `Self-hosted`

## 🚀 O que estou construindo

### 🔷 [FlexCore](https://github.com/marcossancal/FlexCore) — *Open Source · Manutenção Ativa*
> **Uma plataforma backend low-code, auto-hospedada, construída em PHP.**

Pense no Airtable — mas no seu próprio servidor, com uma API REST completa, motor de automações, sistema de plugins e controle de acesso por papéis. Com o FlexCore você cria entidades dinâmicas (tabelas), define campos de 30 tipos, gerencia registros por uma interface web e expõe tudo via API — **sem escrever uma linha de SQL ou código backend**.

Feito para desenvolvedores que querem lançar MVPs rapidamente sem abrir mão do controle.

`PHP` · `MySQL/MariaDB` · `REST API` · `Open Source` · `Self-hosted`
